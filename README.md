### CSS and Documents 

# Elements 
  - HTML building blocks styled using CSS.
# Replaced Elements 
  - Elements replaced by external content (img, video).
# Non-Replaced Elements 
  - Elements whose content comes from HTML (div, p).
# Element Display Roles 
  - Controls layout behavior (block, inline, none).
# CSS + HTML
  - CSS connects to HTML via external, internal, or inline styles.
# <link> Tag
  - Links external CSS files to HTML.
# <style> Element 
  - Defines internal CSS inside HTML.
# @import Directive 
  - Imports one CSS file into another.
# HTTP Linking 
  - Loads stylesheets from external URLs or CDNs.
# Inline Styles 
  - CSS written directly inside HTML tags.
# Stylesheet Contents 
  - CSS rules written using selectors and properties.
# Rule Structure 
  - Selector + property + value format.
# Vendor Prefixing 
  - Browser-specific CSS properties.
# Whitespace Handling 
  - Extra spaces donâ€™t affect CSS behavior.
# CSS Comments 
  - Used to document CSS code.
# Media Queries 
  - Apply styles based on screen size or device.
# Media Types 
  - Target screen, print, or all media.
# Media Descriptors 
  - Conditions like width and orientation.
# Feature Queries 
  - Apply styles only if a feature is supported.
# Summary 
  - CSS enables responsive, maintainable, and scalable design.
